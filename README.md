项目由中间件和API管理界面组成

中间件功能:  
1、路由转发：接收请求，转发到后端的每个服务上去。  
2、过滤器：在服务网关中可以完成一系列的横切功能，例如权限校验、负载均衡、限流以及监控等  
3、日志收集。
